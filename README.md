# Oriz Slice — PDF tools

> Merge, split, compress, sign, redact, OCR — a free PDF toolkit that runs entirely in your browser.

**Live at**: https://pdf.oriz.in · **Status**: production

## What this is

A browser-based PDF toolkit with 30+ tools. Every operation runs client-side — your files never leave your device. Sign-in is optional and only powers cross-app features.
